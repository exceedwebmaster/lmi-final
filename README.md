# LMI Website

## About LMI

LMI Fawn offers precision-engineered shower enclosures, wardrobe doors, mirrors, and custom organizers. With 50+ years of experience, we deliver high-quality, innovative solutions for residential and commercial spaces, combining modern design, functionality, and exceptional service.

---

## Project Overview

This is a static website built using:

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Bootstrap 5

The architecture is structured for scalability, maintainability, and clean separation of concerns.

---

## Core Objectives

- Maintain clean file organization
- Separate reusable components
- Keep assets centralized
- Support easy deployment
- Ensure responsiveness via Bootstrap

---

## Tech Stack

- HTML5 for structure
- CSS3 for styling
- JavaScript for interactivity
- Bootstrap 5 for layout and UI components
